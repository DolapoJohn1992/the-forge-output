# The Forge: Linux Practical Exam
### Cybersecurity Foundations Intensive | The Knowledge House

## 🎯 Project Overview
This repository contains the results of a high-pressure Linux diagnostic and security audit. The mission involved hunting for hidden system logs, extracting forensic evidence, and securing artifacts according to strict permission protocols.

## 🛠️ Technical Tasks Completed
* **Artifact Discovery:** Utilized advanced `find` commands with root privileges to locate hidden system logs.
* **Security Hardening:** Implemented strict read-only permissions (`chmod 444`) on sensitive log files to prevent tampering.
* **Reporting:** Documented the full technical workflow and terminal command history in a structured report.

## 📂 Repository Contents
* `practical_exam_report.txt`: Full audit documentation.
* `forge_alpha.log`: Secured forensic log artifact 1.
* `forge_beta.log`: Secured forensic log artifact 2.
